A grande bomba da semana foi a decisão da Oracle de mover um processo contra o Google, que está sendo acusado de quebrar algumas patentes do Java no sistema operacional para celulares  Android. Quando a Sun, criadora do Java e adquirida pela Oracle no começo do ano, processou a Microsoft também por causa do Java, além da multa bilionária, também conseguiu fazer com que a Microsoft desistisse do Java e criasse sua plataforma .NET. Será que o Google também vai desistir do Java? Se isso acontecer, será que o Java ainda vai ser relevante ou esse movimento vai afastar outras empresas (e desenvolvedores) com medo de também serem processadas? Outra notícia polêmica na semana foi a de que o IronRuby, implementação da Microsoft da linguagem Ruby, que roda sobre o .NET Framework, estaria prestes a ser descontinuado. Um dos principais desenvolvedores do time responsável pelo IronRuby deixou a Microsoft e, segundo ele, só restou uma única pessoa na equipe. Para finalizar, a data de lançamento da versão beta do Internet Explorer 9 foi anunciada e será em 15 de setembro, portanto, daqui há um mês!
